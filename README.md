# Vedashree Enterprises Website

A responsive static website for Vedashree Enterprises, providing RO water purifier service, installation, and repair in Sangamner, Maharashtra.

---

## 🚀 Tech Stack

- HTML5  
- CSS3 (Responsive Design)  
- JavaScript (Vanilla JS)  
- GitHub Pages (Hosting)

---

## 📌 Project Overview

This is a lightweight, mobile-first business website designed to showcase RO services, pricing, AMC plans, and contact options. It is optimized for fast loading and local SEO visibility.
